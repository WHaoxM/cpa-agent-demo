## 一、全部页面功能

### 公共页面

#### 1. 着陆页 `LandingView.vue` — `/`

- 全屏暗色星空粒子动画（星尘、流星、星云、鼠标视差光晕）
- 品牌展示"职导星图 — 找到你在职业星空中的坐标"
- GSAP 打字机标题入场，CTA 按钮跳转登录页

#### 2. 登录页 `LoginView.vue` — `/login`

- 角色切换（学生 / 管理员）+ 账号密码表单
- 快速一键体验（演示账号登录）
- 登录成功获取 JWT Token，写入用户状态，跳转首页
- 底部展示平台四大功能亮点

#### 3. 404 页面 `NotFoundView.vue`

- 访问不存在的路由时展示

### 应用主框架

#### `AppLayout.vue` — `/app/*`

### 学生端页面（核心业务）

#### 4. 首页仪表盘 `HomeCenter.vue`

**路由**：`/app/dashboard`

- **Hero 区**：问候语、日期、目标岗位标签、进行中课程数
- **智能引导卡**：根据用户职业规划完成阶段（4 步），自动推荐下一步行动并跳转
- **学习概览**：已完成课程数、连续学习天数、平均测评分
- **职业规划路径**：4 步进度可视化（探索市场 → 了解自己 → 提升技能 → 沉淀成果），展示各阶段完成情况
- **平台功能卡片**：6 个入口（职业分析、技能自评、职途导航、技能提升、我的报告、AI 助手）
- **管理员视图**：简洁入口（岗位数据集 + 知识库维护）

---

#### 5. 职业分析 `CareerAnalysis.vue`

**路由**：`/app/student/career-analysis`

- **行业领域气泡图**：7 大行业 × 多个岗位子气泡，展示需求指数和薪资水平
- **全国地图热力**：ECharts 地图按岗位展示 34 省需求指数 + 薪资中位数，支持按年份切换
- **全国 KPI 面板**：平均中位薪资 + 岗位需求总量 + 同比增速
- **省份详情**：点击省份 → 薪资箱形图 + 需求趋势折线（支持双省份对比）
- **岗位薪资区间**：初级 / 中级 / 高级三段堆积柱状图
- **AI 市场洞察**：4 条翻页评论（整体趋势 / 供需分析 / 薪资建议 / 发展前景）
- **关注 / 取关**：15 个岗位方向一键关注，写入 targetRoles
- **CTA**："去职途导航"按钮

---

#### 6. 技能自评 `ExamsView.vue`

**路由**：`/app/exams`

- **选方向**：5 大方向（前端 / 后端 / 测试 / 数据 / ML），每方向 3 条细分赛道（共 15 个子方向）
- **答题**：每条赛道 8 项核心技能，5 级自评量表（没听说过 → 精通），映射 0–100 分
- **结果**：各技能得分条形图 + 等级标签 + 整体雷达，评估结果同步更新用户能力画像
- 完成后可带参数跳转 CareerAnalysis 查看对应方向数据

---

#### 7. 职途导航 `CareerNavigation.vue`

**路由**：`/app/student/career-navigation`

- **简历导入**：拖拽上传文件（doc / pdf / ppt / txt）或粘贴文字
- **偏好方向预选**：5 大领域折叠卡片，展开显示子岗位
- **"没有简历？"** → 引导跳转 ResumeBuilder
- **Agent 4 阶段流式分析**：
  1. 文档预处理 + 技能抽取 + 岗位推断
  2. 七维能力评估（专业技能 / 证书 / 实习可计算，创新 / 学习 / 抗压 / 沟通由后端论文算法计算）
  3. 经历分析（项目、荣誉、实习亮点提取）
  4. AI 综合评语（支持流式逐字输出）
- **右侧 Agent Dashboard**：实时 Trace 日志、6 步阶段进度条、当前步骤标签
- **左侧进度面板**：与 Dashboard 共享同一 session 时间线
- 完成后 Dashboard 收缩，左侧展示嵌入式 **TalentPortrait** 能力画像
- 解析结果（技能列表、匹配度、洞察）写入全局状态，供其他页面消费

---

#### 8. 个人能力画像 `TalentPortrait.vue`

**路由**：`/app/student/career-portrait`

- 支持**独立访问**和**嵌入模式**（由 CareerNavigation 嵌入）
- **4 阶段渐进渲染**：解析 → 评估 → 分析 → 综评，每个阶段随 Agent 事件逐步揭示
- **个人信息卡**：姓名、学校、专业、年级、目标岗位
- **简历完整度**：百分比进度条
- **六维能力雷达**：D3RadarChart 可视化各维度得分
- **竞争力评分**：综合分值 + 等级
- **技能标签云**：按权重 / 分类展示
- **荣誉 & 项目经历**：卡片列表
- **AI 综合评价**：逐字打字机效果，支持 ReadableStream 真流式
- 可保存画像到报告库

---

#### 9. 职业能力图谱 `CareerAbilityShell.vue`

**路由**：`/app/student/career-ability`

- **Shell 容器**管理 3 种子视图切换：
  - **Graph**：ECharts 同心圆轨道动画图谱
  - **Dual**：双栏对比（图谱 + 岗位介绍）
  - **Workspace**：工作台模式（图谱 + 岗位介绍 + Agent 日志流逐步显现节点）
- 展示岗位的能力关系网（节点分组：专业 / 岗位 / 认知 / 通用技能）
- 已掌握技能高亮（来自简历解析结果）

---

#### 10. 课程体系 / 技能图谱 `CourseSystemGraph.vue`

**路由**：

- Focus 模式：`/app/student/course-system?role=xxx`（单职业方向）
- Overview 模式：`/app/student/skill-graph`（15 个岗位总览）

**功能**：

- **Three.js 3D 图谱**：基础 → 进阶 → 高阶 → 岗位 四层节点，层间有依赖边
- OrbitControls 相机交互 + GSAP 动画
- 层级高亮切换、标签显隐、hover 展示前后层关联节点
- 选中岗位节点 → 跳转 LearningCenter

---

#### 11. 技能提升 `LearningCenter.vue`

**路由**：`/app/student/learning`

- 从 `?role=xxx` 读取岗位，按 skillTags ∩ keySkills 匹配推荐课程
- **课程列表**：搜索、分类筛选、分页
- **D3 Treemap**：课程分类面积可视化
- **D3 ChordDiagram**：技能关联关系弦图
- 课程卡片展示进度、操作按钮
- 记录学习时长和完成状态
- 可返回课程体系图谱

---

#### 12. 职业生涯发展报告 `CareerReport.vue`

**路由**：`/app/student/career-report`

- **分析模式**：
  - 目标方向选择（来自关注列表或简历解析预测）
  - 7 维能力雷达图对标（专业技能 / 证书 / 创新 / 学习 / 抗压 / 沟通 / 实习）
  - 岗位成长路径图（GSAP MotionPath 攀登动画），展示路径边技能差距
  - Agent 生成个性化成长计划（分阶段：phase / goal / tasks / milestone）
- **报告模式**：
  - 可编辑富文本报告
  - AI 一键润色 + diff 对比
  - 内容完整性检查
  - 导出 PDF / Word
- 保存报告到报告库

---

#### 13. 快速制作简历 `ResumeBuilder.vue`

**路由**：`/app/student/resume-builder`

- 仿 magicv.art **三栏布局**（左模块导航 / 中表单 / 右 A4 实时预览）
- **7 个模块**：基本信息、专业技能、工作经验、项目经历、教育经历、实习 & 荣誉、自我评价
- Tiptap 富文本编辑器（自我评价模块）
- 头像上传、6 色主题切换
- **AI 一键生成**：各分区内容由 Agent 根据上下文生成
- **PDF 导出** + **云端多份简历管理**

---

#### 14. 我的报告 `MyReports.vue`

**路由**：`/app/student/my-reports`

- **3D 书架场景**（BookshelfScene）：两组书架
  - 个人能力画像书架
  - 职业生涯发展报告书架
- 点击书本 → BookOpenOverlay 浮层展开报告详情
- 支持删除报告
- 快捷入口："开始测评" → CareerNavigation，"生成报告" → CareerReport

---

#### 15. 心仪岗位 `Favorites.vue`

**路由**：`/app/student/favorites`

- 数据来源：在 CareerAnalysis 中关注的职业方向
- 每个岗位卡片展示：市场数据（中位薪资、需求量、增速）、匹配度评分（来自 Agent 解析产出）、核心技能标签（可展开）
- **排序**：匹配度 / 薪资 / 关注时间
- **匹配状态**：未评估 / 匹配度低 / 已评分
- "进行人岗匹配" → 跳转 CareerNavigation
- 取消关注

#### 16. AI 助手 `AIAssistant.vue`

- **真实 AI 对话**：上下文携带用户画像 + 学习进度，SSE 流式输出
- **思考过程展示**：展开 / 折叠思考步骤 + 思考耗时

## 三、关键跨页面联动

1. **CareerAnalysis → Favorites**：关注方向写入 targetRoles，Favorites 读取展示市场数据 + 匹配度
2. **CareerNavigation Agent → 全局**：解析产出的技能列表、匹配度、洞察写入 resumeStore，供 Favorites（匹配度）、CareerAbility（已掌握高亮）、CareerReport（能力数据）消费
3. **ExamsView → 能力画像**：自评结果更新用户能力数据，影响后续 Agent 评估和报告对标
4. **CourseSystemGraph → LearningCenter**：选中岗位节点跳转，按岗位过滤推荐课程
5. **CareerNavigation ↔ TalentPortrait**：共享 usePortraitSession 时间线，嵌入式渐进渲染
6. **CareerNavigation → ResumeBuilder**："没有简历？"入口引导制作
7. **TalentPortrait / CareerReport → MyReports**：保存画像 / 报告，3D 书架统一展示
8. **HomeCenter 智能引导卡**：根据各步骤完成状态自动指向下一步对应页面
9. **AIAssistant 全程辅助**：上下文携带用户画像 + 学习进度，回复中嵌入页面跳转引导
