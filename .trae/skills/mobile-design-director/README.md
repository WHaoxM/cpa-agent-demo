# mobile-design-director

> **v5.0** — Just describe what you need, or share a screenshot. Get formula-based design schemes.

## What it does

A mobile UI/UX Design Director skill for AI coding agents. Provides:

- **Formula-based design schemes** — each recommendation is a visible formula like `[1px Inner Border] + [Noise Grain] + [Inner Glow] = Premium Container`
- **Mandatory requirements review** — always thinks holistically before designing
- **Multi-scheme output** — 2-3 approaches targeting different design dimensions
- **Implementation specs** — exact values, code snippets, ready to ship

## How to use

Just describe what you need in natural language:

```
"帮我设计一下这个页面"
"看看这个截图，怎么做更好看"
"redesign this component to feel more premium"
"设计整个功能的品牌体验"
```

Or provide a screenshot and ask for review/redesign.

## The Workflow

```
Step 0: 需求全景 — Review requirements holistically (MANDATORY)
Step 1: 诊断 — Analyze what exists
Step 2: 配方 — Build 2-3 formula-based schemes
Step 3: 裁决 — Professional recommendation + implementation spec
```
